## 实验结果

目前只完成了**Helmet**, **Head**两个类别的检测，**person**由于只有116个训练数据标注了目标，因此没有办法很好的训练，目前一个相对可行的思路是训练一个比较差的**person**检测器，并用这个检测器检测训练集，然后与**helmet**以及**head**标签计算IOU，与这两个标签交叠区域较大的**person**伪标签记为质量较差的真实标签，然后反复迭代。
